# ZooKeeper Docker Container for Kubernetes


## For Developers

#### To build

```sh
## From Container directory
pwd
.../kube-zookeeper-statefulsets/container


./build.sh
```

#### To debug with a bash prompt  

```sh
## From Container directory
pwd
.../kube-zookeeper-statefulsets/container


util/debug.sh
```

```sh
## From Container directory
pwd
.../kube-zookeeper-statefulsets/container


util/push.sh
```

#### To run   

```sh
## From Container directory
pwd
.../kube-zookeeper-statefulsets/container


util/run.sh
```
